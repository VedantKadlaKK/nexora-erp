# Nexora ERP ⚙️

Nexora ERP is a next-generation logistics and freight forwarding ERP platform focused on realtime operational control and event-driven architecture.

## Features

- Realtime shipment updates
- Finance risk synchronization
- Activity logging system
- JWT authentication
- PostgreSQL database architecture
- FastAPI backend
- React + Tailwind frontend
- Socket.IO realtime communication
- Event-driven ERP logic

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- Socket.IO Client

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- JWT Authentication
- Socket.IO

## Architecture

Nexora ERP demonstrates:

- operational event propagation
- interconnected enterprise modules
- realtime synchronization
- enterprise backend architecture
- full-stack system design

## Example ERP Flow

When shipment status changes:
- shipment table updates
- finance risk updates
- activity logs update
- realtime dashboard syncs

without refreshing the page.
